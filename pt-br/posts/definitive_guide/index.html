O guia para iniciantes em Ciências de Dados

April 1, 2021

Iniciarei uma série de posts com todo o material que encontro sobre Data Science. Este primeiro post focarei em um público específico pessoas que ainda estão descobrindo a área e aprendendo o básico. Todo conteúdo postado aqui está em inglês, alguns possuem legendas e traduções, mas a grande maioria não. Existem algumas bibliotecas importantes para se aprender, no entanto, como não vou focar em uma linguagem de programação específica, o próprio leitor ao explorar o material aqui irá encontrar diversas bibliotecas interessantes tanto para Python como para R (linguagens mais populares).

O que você precisa saber antes de entrar em da Data Science

Gosto de pensar que a ciência de dados é uma carreira de programação, porque na maior parte do tempo você estudará sobre seus dados e o problema dentro deles, ou irá manipular, construir modelos de aprendizado de máquina (ML) não consome muito tempo e, para ser sincero, na maioria das vezes é uma coisa de “experimentar e tentar”, no entanto ML tem uma teoria matemática complexa, e se você quiser ser um bom cientista de dados precisará desenvolver ao menos uma compreensão sobre o campo. Então antes de qualquer coisa gosto de listar o básico do básico, a linguagem de programação mais usada é python e R, escolha a que te fazer sentir mais confortável.

Este conteúdo é bastante fácil de encontrar na internet, porém postarei algumas coisas que considero bem produtivas aqui.

Programação

É importante que além de estudar o básico de Python ou R, você também tenha em mente boas práticas de programação, aconselho também a desenvolver uma familiaridade com Programação Orientada a Objetos, principalmente para conteúdo de Deep Learning mais avançado. Eu também aconselharia você a aprender SQL porque é uma linguagem de programação fácil e muito importante para o mercado.

Cálculo Básico

Os principais tópicos a aprender aqui são: funções, limites, derivadas, continuidade de funções, otimização de funções e Integral. Tenha em mente que a parte importante é saber o que é uma derivada, o que é um limite, o que você faz para encontrar o ponto de otimização de uma função e coisas do tipo, e não se trata de encontrar a derivada / integral de alguma função enorme

Alguns posts podem aconselhá-lo a simplesmente ignorar a teoria por trás disso e pular para tutoriais que ensinam como fazer algumas coisas realmente básicas, como:

from sklearn.ensemble import RandomForestClassifier
import pandas as pd
df = pd.read_csv('dataset_lindo_e_bonito_que_nunca_vai_existir_na_vida_real.csv')
X, y =  df.iloc[:,:-1], df.iloc[:, -1]
clf = RandomForestClassifier()
clf.fit(X, y) #Funciona!!!

No entanto, uma parte importante do trabalho é entender como começar com um banco de dados confuso e desenvolver o produto final. Esse tipo de tutorial pode ser útil para ensinar sobre codificação, mas é muito superficial sobre a solução de problemas (que é o assunto da Data Science). Na minha opinião, pode ser muito mais útil conhecer cerca de 2 a 3 modelos e ter uma “caixa de ferramentas de ML” que você pode usar para compreender um problema e desenvolver alguma solução bem arredondada.

Introdução a Ciência de dados

Depois de saber o que é um teste estatístico e como funciona a otimização de uma função, você pode começar com este material.

An Introduction to Statistical Learning

Este é provavelmente o melhor livro para introduzir alguém no assunto (e é o meu favorito), ele tem muitos recursos online. O livro se concentra principalmente na aplicação em R, mas eu sugiro fortemente que você continue com este livro mesmo se você preferir Python.

  1. O livro você encontra aqui
  2. Dados e códigos usados no livro
  3. Aulas online com os próprios escritores

Data Science from Scratch: First Principles with Python (Joel Grus)

Este livro implementa uma série de algoritmos de ML do ZERO, não é exatamente um livro de introdução, mas o ajudará a melhorar suas habilidades de programação enquanto ensina algumas noções básicas de ciência de dados. Infelizmente, não há recursos online para este livro, apenas certifique-se de pesquisar e procurar bastante pelo pdf!

  1. Machine Learning
  2. Todos os cursos sobre Deep learning da Deeplearning.ai

Kaggle

Kaggle é uma comunidade gigantesca de ciência de dados e você pode encontrar os mais diversos conteúdos lá, desde um guia para iniciantes até técnicas avançadas muito específicas. E eles TAMBÉM têm alguns cursos úteis explicando o básico de um tema complexo.
