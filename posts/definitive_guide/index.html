The guide to the absolute Beginner in Data Science

April 5, 2021

Do you want to become a Data Scientist? In this series I’ll cover the basics to study data science here.

What you need to know before going in Data Science’s bandwagon

I like to think Data Science as a programming carrer, because most of your time you’ll be either learning about your data and the problem within it or you’ll be manipulating it, building machine learning (ML) models is not time consuming, and to be fair most of the time is an “experiment and try” thing, however ML has a deep mathematical theory behind it, and if you want to be a good data scientist you’ll need to develop a fair comprehension about the field. So before anything I like to list the basic of the basics, just remembering that the most used programming language is python and R, as I am more proeficient with python I’ll be posting resources for this language.

This content is fairly easy to find on the internet, however I’ll post some good stuff to study.

Programming

It’s important for you to focus on Python or R, aside from basic progrmaming you’ll also need to have on mind good programming practices and know some of Object Oriented Programming for more advanced Deep Learning content. I would also advise you to learn SQL because’s an easy programming language but a important skill to have.

Linear Algebra

This content is fairly easy to find on the internet The main topics to learn here are: vectors, matrices and basic operations, factorization, transposes and inverses, projections, orthogonal vectors and matrices, determinants, eigen vectors and eigen values, matrice inverse.

Basic Calculus

The main topics to learn here are: functions, limits, derivatives, function continuity, function optimization and Integral. Keep in mind that the important part is knowing what a derivative is, what is a limit, what do you do to find a function’s optimization point and stuff like that, and this is not about finding the derivative/integral of some enormous function

Some posts may advise you to just ignore the theory behind it and jump to tutorials teaching you how to do some really basic stuff like:

from sklearn.ensemble import RandomForestClassifier
import pandas as pd
df = pd.read_csv('a_fantastic_dataset_that_will_never_exist_in_real_life.csv')
X, y =  df.iloc[:,:-1], df.iloc[:, -1]
clf = RandomForestClassifier()
clf.fit(X, y) #it works!!!

However one important part of the job is understanding how to start with a messy database and develop final product. This type of tutorials can be useful to teach about coding but is way to shallow about problem solving (which is what Data Science is about). In my opnion it can be way more useful knowing about 2 to 3 models and having a “ML toolbox” that you can use to compreehend a problem and develop some well rounded solution.

Data Science Theory & Application

An Introduction to Statistical Learning

This is probably the best book to introduce someone in the subject (and is my personal favorite), it has a lot of online resources. The book focuses mainly on R but I STRONGLY suggest you to stick with this book even if you prefer Python

  • You can find the book clicking in here
  • Datasets and codes used in the book
  • Online classes by the own writers of this book

Data Science from Scratch: First Principles with Python (Joel Grus)

This book develop a series of ML alghoritms by SCRATCH, it’s not exactly an introduction book, but’ll help you to improve your programming skills while teaching some basic of data Science. Unfortunately there’s no online resources for this book, just make sure to search it right!

Coursera

An online plataform founded by Daphne Koller and Andrew Ng which is one of the most influential names in Data Science community. All courses can be attended free but without a certification. The most important ones are:

  1. How to Win a Data Science Competition: Learn from Top Kagglers
  2. Machine Learning
  3. The whole series of Deep Learning offered by Deeplearning.ai

Kaggle

Kaggle is a gigantic data science community and you can find the most diverse content there, from beginner guide to very specific advanced techniques. And they ALSO have some useful courses

Conclusion
